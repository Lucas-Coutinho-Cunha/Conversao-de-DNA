<---------- CONVERSOR DE DNA PARA RNA ---------->


Pasta assets:
    dna1.png            -> Decoração na esquerda do cabeçalho do site;
    dna2.png            -> Decoração na direita do cabeçalho do site;
    logo-uff.png        -> Logotipo da faculdade do projeto no rodapé;
    bar.png             -> Barra separadora usada na tabela de códons;
    gib2c.png           -> Logo da organização responsável pelo projeto;
    favicon2.ico        -> Ícone mostrado na aba do site;
    Nunito-Regular.ttf  -> Fonte utilizada no site, baixada do fonts.google.com;


Pasta css:
    reset.css   -> Arquivo CSS que padroniza a aparência do site em todos os navegadores.
                   Isso é necessário pois, por padrão, alguns navegadores incluem barras
                   brancas ou margens que não queremos no site.
    style.css   -> Arquivo CSS onde toda a aparência do site é codificada.


index.php       -> Código do site, incluíndo o HTML que determina sua estrutura e o
                   backend escrito em PHP. O código converte o input de DNA para RNA
                   e depois converte cada códon RNA para sua respectiva proteína.
